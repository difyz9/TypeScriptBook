## 🔍 常见问题记录

### 第 1 章常见问题
**Q: TypeScript 和 JavaScript 的关系是什么？**
A: TypeScript 是 JavaScript 的超集，提供静态类型检查和现代 ECMAScript 特性。

**Q: 为什么要使用 TypeScript？**
A: 提高代码质量、减少运行时错误、改善开发体验、支持大型项目维护。

### 第 2 章常见问题
**Q: any 和 unknown 的区别？**
A: any 会跳过类型检查，unknown 需要类型断言或类型保护才能使用。

**Q: 什么时候使用类型断言？**
A: 当你比 TypeScript 更清楚某个值的类型时，但要谨慎使用。

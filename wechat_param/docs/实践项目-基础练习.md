## 1. 个人名片小程序

### 项目描述
创建一个展示个人信息的数字名片，包含基本信息、技能展示、联系方式等功能。

### 学习目标
- 掌握基本的页面布局
- 学习WXML和WXSS的使用
- 理解数据绑定机制
- 熟悉小程序的基本组件

### 功能要求

#### 基础功能
1. **个人信息展示**
   - 头像、姓名、职位
   - 公司/学校信息
   - 个人简介

2. **技能展示**
   - 技能列表
   - 技能熟练度（进度条）
   - 技能分类

3. **联系方式**
   - 电话、邮箱、微信
   - 地址信息
   - 社交媒体链接

4. **作品展示**
   - 项目列表
   - 项目详情
   - 图片展示

### 技术要点

#### 页面结构
```
pages/
├── index/          # 主页面
├── skills/         # 技能展示页
├── works/          # 作品展示页
└── contact/        # 联系方式页
```

#### 核心技术
- 基础组件使用（view, text, image, button）
- Flex布局应用
- 数据绑定和事件处理
- 页面跳转和参数传递

## 2. 计算器小程序

### 项目描述
实现一个功能完整的计算器，支持基本四则运算、科学计算等功能。

### 学习目标
- 掌握事件处理机制
- 学习复杂的业务逻辑实现
- 理解状态管理
- 熟练使用条件渲染

### 功能要求

#### 基础功能
1. **基本运算**
   - 加、减、乘、除
   - 小数点运算
   - 清除功能

2. **界面交互**
   - 按钮点击效果
   - 结果显示
   - 错误处理

#### 进阶功能
1. **科学计算**
   - 开方、平方
   - 三角函数
   - 对数运算

2. **历史记录**
   - 计算历史保存
   - 历史记录查看
   - 重用历史结果

#### 逻辑实现
```javascript
// pages/calculator/calculator.js
Page({
  data: {
    displayValue: '0',      // 显示的数值
    expression: '',         // 当前表达式
    previousValue: 0,       // 上一个操作数
    currentOperator: null,  // 当前运算符
    waitingForOperand: false, // 是否等待新的操作数
    history: []            // 计算历史
  },

  // 输入数字
  inputNumber(e) {
    const number = e.currentTarget.dataset.number;
    
    if (this.data.waitingForOperand) {
      this.setData({
        displayValue: number,
        waitingForOperand: false
      });
    } else {
      const displayValue = this.data.displayValue === '0' ? number : this.data.displayValue + number;
      this.setData({
        displayValue: displayValue
      });
    }
  },

  // 输入小数点
  inputDot() {
    if (this.data.waitingForOperand) {
      this.setData({
        displayValue: '0.',
        waitingForOperand: false
      });
    } else if (this.data.displayValue.indexOf('.') === -1) {
      this.setData({
        displayValue: this.data.displayValue + '.'
      });
    }
  },

  // 输入运算符
  inputOperator(e) {
    const operator = e.currentTarget.dataset.operator;
    const inputValue = parseFloat(this.data.displayValue);

    if (this.data.previousValue === null) {
      this.setData({
        previousValue: inputValue
      });
    } else if (this.data.currentOperator) {
      const currentValue = this.data.previousValue || 0;
      const newValue = this.performCalculation(currentValue, inputValue, this.data.currentOperator);

      this.setData({
        displayValue: String(newValue),
        previousValue: newValue
      });
    }

    this.setData({
      waitingForOperand: true,
      currentOperator: operator,
      expression: `${this.data.previousValue} ${this.getOperatorSymbol(operator)}`
    });
  },

  // 执行计算
  calculate() {
    const inputValue = parseFloat(this.data.displayValue);

    if (this.data.previousValue !== null && this.data.currentOperator) {
      const newValue = this.performCalculation(this.data.previousValue, inputValue, this.data.currentOperator);
      
      // 保存到历史记录
      const historyItem = {
        expression: `${this.data.previousValue} ${this.getOperatorSymbol(this.data.currentOperator)} ${inputValue}`,
        result: newValue,
        timestamp: new Date().toLocaleString()
      };

      this.setData({
        displayValue: String(newValue),
        expression: '',
        previousValue: null,
        currentOperator: null,
        waitingForOperand: true,
        history: [historyItem, ...this.data.history.slice(0, 9)] // 保留最近10条记录
      });
    }
  },

  // 执行具体的计算操作
  performCalculation(firstValue, secondValue, operator) {
    switch (operator) {
      case '+':
        return firstValue + secondValue;
      case '-':
        return firstValue - secondValue;
      case '*':
        return firstValue * secondValue;
      case '/':
        return secondValue !== 0 ? firstValue / secondValue : 0;
      default:
        return secondValue;
    }
  },

  // 获取运算符显示符号
  getOperatorSymbol(operator) {
    const symbols = {
      '+': '+',
      '-': '-',
      '*': '×',
      '/': '÷'
    };
    return symbols[operator] || operator;
  },

  // 清除
  clear() {
    this.setData({
      displayValue: '0',
      expression: '',
      previousValue: null,
      currentOperator: null,
      waitingForOperand: false
    });
  },

  // 退格
  backspace() {
    const displayValue = this.data.displayValue;
    if (displayValue.length > 1) {
      this.setData({
        displayValue: displayValue.slice(0, -1)
      });
    } else {
      this.setData({
        displayValue: '0'
      });
    }
  },

  // 百分号
  percentage() {
    const value = parseFloat(this.data.displayValue) / 100;
    this.setData({
      displayValue: String(value)
    });
  }
});
```

## 3. 待办事项小程序

### 项目描述
创建一个功能完善的待办事项管理工具，支持任务的增删改查、分类管理、提醒功能等。

### 学习目标
- 掌握本地存储的使用
- 学习列表操作和状态管理
- 理解组件间通信
- 熟练使用模板和自定义组件

### 功能要求

#### 核心功能
1. **任务管理**
   - 添加新任务
   - 编辑任务内容
   - 删除任务
   - 标记完成/未完成

2. **任务分类**
   - 按优先级分类
   - 按类别分类
   - 自定义标签

3. **数据持久化**
   - 本地存储任务数据
   - 数据导入导出
   - 数据同步

### 实现示例

#### 数据结构设计
```javascript
// 任务数据结构
const taskSchema = {

  category: 'work',          // 分类：work/personal/study

  createdAt: '2025-01-01',   // 创建时间

};
```
